# Tree Encoded Bitmaps

This is a Go implementation of Tree Encoded Bitmaps, as described by [this
paper](https://db.in.tum.de/~lang/papers/tebs.pdf). I've previously used
[Roaring Bitmaps](https://roaringbitmap.org/) and was curious to see how this
approach compared. It's also an excuse to play with
[AntiGravity](https://antigravity.google/) a bit. Needless to say, this should
not be used in production.
